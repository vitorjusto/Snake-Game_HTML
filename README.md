# Snake-Game_HTML

ATUALIZAÇÃO (22/09/2021):

CORREÇÕES DE BUGS;

A comida não gera mais dentro da cobra;

correção de teleporte para comida (quando a comida estiver no canto e teleportar para ela, antes a cobra não comia);
